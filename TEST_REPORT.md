🚀 КАК ИСПОЛЬЗОВАТЬ:

1. УСТАНОВИТЬ ЗАВИСИМОСТИ:
   ```bash
   pip install -r requirements.txt
   ```

2. ЗАПУСТИТЬ ПРИЛОЖЕНИЕ:
   ```bash
   python app.py
   ```

3. ОТКРЫТЬ АДМИН-ПАНЕЛЬ:
   - Перейти на http://localhost:5000
   - Залогиниться как администратор
   - В верхней части видно уведомление об обновлении

4. ПРОВЕРИТЬ ОБНОВЛЕНИЕ:
   - Нажать "Обновить сейчас"
   - Подтвердить в диалоге
   - Приложение загружает обновления
